#  Python Calculator Developed By theycreate(ABDUL BASIT)

A command-line calculator built using Python that performs basic arithmetic operations with proper input validation and error handling.

## Features
- Addition, Subtraction, Multiplication
- Division & Modulus with zero-division protection
- Input validation for numbers and operators
- Calculation history saved to a file
- User-friendly CLI interface

## Technologies Used
- Python 3
